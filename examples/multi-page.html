Introduction

This document demonstrates how to create multi-page PDFs from HTML using the html2pdf tool. Each section is designed to appear on a separate page when converted to PDF.

Key Features

The html2pdf tool preserves the following elements across multiple pages:

  • Page breaks at specified locations
  • Consistent formatting and styling
  • Background colors and gradients
  • Custom page sizes and orientations
  • Headers and footers (when enabled)

Important Note

Technical Implementation

CSS for Page Breaks

To create multi-page documents, use CSS page break properties:

.page { page-break-after: always; } .page:last-child { page-break-after: auto; } .keep-together { page-break-inside: avoid; }

Command Line Usage

Convert this multi-page document with the following command:

html2pdf multi-page.html -o multi-page.pdf -f A4 --margin-top 20mm

You can also add headers and footers with page numbers:

html2pdf multi-page.html --header --footer
